# House-Price-Prediction
Machine Learning project that predicts house prices based on property features such as area, bedrooms, bathrooms, stories, and parking using Python and Scikit-Learn.
## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- House price prediction
- User-friendly implementation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Flask

## Dataset Features
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Price

## Machine Learning Model
- Linear Regression
